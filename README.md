# VehicleIQ 🚗📊

### AI-Powered Vehicle Registration Analytics Platform

VehicleIQ is a GenAI-powered analytics platform that enables users to explore and analyze vehicle registration datasets using natural language queries. Instead of writing complex data analysis code, users can simply ask questions in plain English and receive data-driven insights along with interactive visualizations.

---

## 📌 Overview

VehicleIQ combines Generative AI, data analytics, and visualization techniques to make transportation data accessible to both technical and non-technical users. The platform interprets user intent, processes large datasets, executes relevant analyses, and presents the results through meaningful charts and summaries.

The project is built using vehicle registration data from Indian states and union territories, enabling trend analysis across years, regions, and vehicle categories.

## 📊 Dataset

The project uses the **State/UT-wise Registered Motor Vehicles in India (2006–2016)** dataset.

Dataset attributes include:

* State / Union Territory
* Year
* Vehicle Category
* Number of Registered Vehicles

---

## ⚙️ How It Works

1. User enters a question in natural language.
2. Gemini interprets the user's intent.
3. The system converts the query into data operations.
4. Pandas processes the dataset.
5. Visualizations and insights are generated automatically.
6. Results are displayed through charts and summaries.

---

## 📈 Example Queries

* Which state had the highest number of registered vehicles in 2016?
* Show the growth trend of motor vehicles in Maharashtra.
* Compare vehicle registrations across states.
* Display category-wise vehicle distribution.
* Which year saw the highest increase in registrations?
